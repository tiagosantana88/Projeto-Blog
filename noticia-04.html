Microsoft deve oficializar compra do GitHub nesta segunda por US$ 7.5 bi

A Microsoft, mesmo com fama de ser avessa ao Open Source, já tem feito várias contribuições ao próprio GitHub.



A Microsoft teria comprado a GitHub Inc, conforme informações da Bloomberg, sendo assim, a companhia será dona do site de projetos de programação, o que não deve animar muito os desenvolvedores.



Ao que tudo indica, a oficialização do negócio deverá acontecer nesta segunda-feira (4). Com a confirmação da compra, a Microsoft ficará ainda mais próxima da comunidade de código livre, o que poderia não ser viável há alguns anos, porém, agora, está cada vez mais comum.



A Microsoft, mesmo com fama de ser avessa ao Open Source, já tem feito várias contribuições ao próprio GitHub, assim sendo, a negociação pode não ser algo realmente surpreendente.



O valor da compra não foi divulgado, porém, a GitHub foi avaliada em US$ 2 bilhões em 2015, sendo assim, há quem diga que atualmente a empresa já valha o dobro desse valor. O site conta com mais de 27 milhões de desenvolvedores espalhados pelo mundo tido, que criaram mais de 80 milhões de códigos.



Para completar, eles possuem uma área direcionada para empresas, a GitHub Enterprise, que é responsável pela metade dos lucros anuais vendendo códigos, suporte e hospedagem para as empresas, bem como outro plano pago que deixam privados os códigos dos clientes.

Ainda não podemos ter certeza sobre o futuro da GitHub, já que existe o risco da Microsoft perder usuários ou ainda ou mesmo não saber lidar com a aquisição.



De qualquer forma, a Microsoft deverá conquistar a confiança dos desenvolvedores e convence-los a não mudarem de plataforma. O serviço, vale mencionar, recentemente, comemorou 10 anos de existência.

Atualização:


Conforme já mencionado, a Microsoft confirmou nesta segunda-feira (4) a compra do GitHub por um valor de US$ 7,5 bilhões.



Vale mencionar que mesmo antes da notícia ser confirmada, serviços concorrentes ao GitHub viraram alvo de uma migração em massa. Isso indica que milhares de desenvolvedores não aceitaram muito bem a aquisição.
